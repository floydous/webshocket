######################
Welcome to Webshocket!
######################

**Webshocket** is a modern, `asyncio`-based Python library designed to simplify the creation of WebSocket clients and servers. It provides a high-level, object-oriented API that handles the common boilerplate, letting you focus on your application logic.

This library is for you if you want:

* An intuitive, object-oriented API for managing connections and state.
* Built-in best practices like automatic resource management and client reconnection.
* Advanced features like broadcasting and channels made simple.
* A secure, type-safe foundation for your real-time application.

   This documentation is for version |release|. For the source code, please visit the `GitHub repository <https://github.com/your-username/webshocket>`_.
